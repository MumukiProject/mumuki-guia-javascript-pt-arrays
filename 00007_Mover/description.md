> Declara um procedimento `mover`, que pegue dois arrays, remova o último elemento do primeiro e adicione-o ao segundo.

> Exemplo:

> ``` javascript
> let umArray = [1, 2, 3];
> let outroArray = [4, 5];
 
> mover(umArray, outroArray);
 
> umArray //deveria ser [1, 2]
> outroArray //deveria ser [4, 5, 3]
> ```
